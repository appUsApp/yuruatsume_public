2. プロジェクト概要

『ゆるあつめ』は、ビーチを指でなぞって見つけて・集めて、あなただけのギャラリーを完成させる癒しのコレクションゲームです。
朝・昼・夕・夜と移ろう時間帯で背景とBGMがシームレスに遷移し、出現物やイベントが変化します。
シンプル操作と短いプレイサイクルを重視し、数分で小さな達成感を積み重ねられる設計にしています。
